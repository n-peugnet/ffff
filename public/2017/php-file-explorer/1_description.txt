<p>
	PHP File Explorer est un explorateur de fichier réalisé rapidement en PHP.
</p>
<p>
	Le but de ce projet était de mettre en application mes cours sur les objets inclus et les fonction récursives.
	Il permet de visualiser et d'ouvrir les fichiers d'une arborescence donnée.
	Le chemin absolu n'est jamais révélé et tous les chemin visibles sont relatifs à la racine parametrée dans le fichier config.php.
	De nouvelles fonctions seront probablement ajoutées plus tard.
</p>
<p>
	Le projet est visible sur <a href="https://github.com/n-peugnet/phpFileExplorer" target="_blank">Github</a>
</p>
